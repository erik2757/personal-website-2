Puppy Bowl

Designed and created a whole front-end for a competition using the CRUD method. This website was created to manage the data for people to view the puppy competitors and their owners. You can view all the players in the homepage, view more details, or delete a player. In the header, you can add a new player to the list if you want to add more competition. If any information inputted is incorrect we can always adjust it. Key technologies used in this project include React, CSS, and Express.
